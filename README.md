# Madrid Shopping
Madrid Shopping App IOS with Swift 4 for the final proyect of Advance IOS Development Module.


- This Proyect is writting with Xcode 9 And Swift 4
- Use Core Data for cache Data and images
- Data in Spanish and English
- GCD to read Internet data
- MapKit
- Use of animations inside the App
- Offline
- Use of clean architecture (interactor, managers etc)
- Testing model and DAOs. Develop the app using TDD
- Control internet connection
- We use Constrains in the UI.
